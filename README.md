# Spaceship Merge

**Spaceship Merge** is an Android merge game built in **Android Studio** for medium-sized phone devices. Players combine spaceships to progress, rack up points, and compete for the top spot on the leaderboard.

---

## Gameplay Overview

* Merge identical spaceships to create higher-level ships
* Strategic placement and timing to maximize score
* Increasing difficulty as the board fills
* Competitive scoring to encourage replayability

---

## Competitive Scoring

The game uses **Firebase** to store and retrieve player scores in real time. This enables:

* Persistent score tracking
* Global leaderboards
* Friendly competition between players aiming for the highest score

---

## Monetization

* Integrated **in-game advertisements** to support the app

---

* **Language:** Java / Kotlin (Android)
